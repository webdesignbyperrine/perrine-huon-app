## 3. Cadence et propriétaire

- **Le 1er de chaque mois** : rafraîchir tout ce dashboard (1h-2h de travail).
- **Le 5 de chaque mois** : envoyer un résumé exécutif d'1 page à soi-même (forces / faiblesses / 3 actions M+1).
- **Trimestriellement** : revoir la liste des 50 requêtes cibles et l'ajuster en fonction des opportunités identifiées dans Search Console (requêtes émergentes).

---

## 4. Workflow recommandé

1. **Lundi 1er** : récupérer les données automatisées (Search Console, GA4, Ahrefs, PageSpeed).
2. **Mardi 2** : test manuel des 10 prompts LLM, capture d'écran archivée.
3. **Mercredi 3** : remplir ce template, calculer les évolutions.
4. **Jeudi 4** : identifier les 3 actions prioritaires du mois suivant (en lien avec `SEO-EXTERNAL-CITATIONS.md`).
5. **Vendredi 5** : décider du contenu à publier ce mois (1 article blog + 2 cross-posts).
