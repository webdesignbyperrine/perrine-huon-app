# Images du Projet

## Images à ajouter dans ce dossier :

### 1. Logo Perroquet
- **Fichier** : `logo-perroquet.png` ou `logo-perroquet.svg`
- **Fond** : Transparent ou blanc
- **Utilisation** : Header (navigation)
- **Dimensions recommandées** : 48x48px minimum

### 2. Photo Portrait
- **Fichier** : `portrait-perrine.jpg` ou `portrait-perrine.png`
- **Utilisation** : Section "À propos"
- **Dimensions recommandées** : 800x800px (carré)
- **Format** : JPG ou PNG
- **Note** : Peut utiliser la photo UV/néon de la 4ème capture d'écran

### 3. Images de projets
Placez vos images de projets dans `public/images/projects/`
- Format : JPG ou PNG
- Dimensions : 1200x800px recommandé

## Structure actuelle :
```
public/
└── images/
    ├── logo-perroquet.png (à ajouter)
    ├── portrait-perrine.jpg (à ajouter)
    └── projects/ (à créer si besoin)
```
